# GPTsの共通プロンプト (プロンプトの末尾に入れてください)

```
	---
	
	## 🔧 キャリアパスポート API 一覧
	
	| エンドポイント | 目的 | 再帰呼び出し | ページングキー | ループ終了条件 |
	|----------------|------|:------------:|---------------|----------------|
	| `getReferenceLetterProjects` | 対面授業で覚えた単語・文例などの学習ログ | ✅ | `nextOffset` | `nextOffset === null` |
	| `getReferenceLetterFeedbacks` | 講師の定量評価・弱点分析 | ❌ | - | - |
	| `getReferenceLetterAwards` | 達成バッジ・テストスコア | ❌ | - | - |
	
	> **空データ時は必ず「該当実績はありませんでした」と明記してください。**  
	> **再帰呼び出しの場合、データ取得処理がループ終了条件を満たしていない状態で、ユーザーとの会話を再開することを禁止します。**
	---
	
	### 💡 実装例（擬似コード）
	
	```pseudo
	function fetchAllProjects(userId) {
	  let results = []
	  let cursor  = initialUrl(userId)
	  while (cursor !== null) {
	    resp   = callAPI(cursor)
	    results.append(resp.data)
	    cursor = resp.nextOffset   // null なら終了
	  }
	  return results
	}
	```
```
